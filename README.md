*Prompt utilizado na criação da landing page:*

Crie uma landing page de portfólio moderna e responsiva utilizando apenas HTML, CSS e JavaScript, separando o projeto em index.html, style.css e script.js. O design deve ser inspirado em portfólios profissionais, com tema escuro, glassmorphism, animações suaves e layout elegante. Apresente minha trajetória como Desenvolvedor Back-end, destacando minha formação no SENAC e na Samsung, experiência na Hapvida, participação em hackathons do Hub Caucaia Valley e projetos como Sistema de Reforço Escolar, Sistema de Gestão Escolar, Sistema de Biblioteca e Pokémon Game. Inclua seções para Sobre, Tecnologias, Experiência, Projetos, Formação, Estatísticas e Contato. Implemente efeitos como digitação no Hero, menu mobile, scroll suave, partículas de fundo, animações ao rolar, botão "voltar ao topo" e cards interativos. O resultado deve ter código limpo, comentado, responsivo e pronto para publicação no GitHub Pages ou Vercel.
